
# JavaScript

자바스크립트 코드는 브라우저로부터 3가지 유형의 객체를 제공받아 활용할 수 있음

- DOM객체
- 코어 객체
- HTML DOM 객체
    - HTML Document Object Model
    - HTML 태그 당 DOM 객체 생성
    - 태그가 출력된 모양이나 컨텐츠 제
<br>

## DomTree

- 루트 - document 객체
- 종류 - HTML 태그 종류만큼
- 태그 당 DOM객체 하나씩 생성
- 태그의 포함관계에 따라 DOM 트리에 부모 자식 관계가 결정됨
<br>

### HTML 태그의 요소

- 엘리먼트 이름
- 속성
- CSS 스타일
- 이벤트 리스너
- 콘텐츠
<br>

### DOM 객체의 구성 요소

- 프로퍼티
    - HTML태그 속성 반영
- 메소드
    - DOM객체의 멤버 함수 → 태그 제어
- 컬렉션
- 이벤트 리스너
    - 태그에 작성된 이벤트 리스너 반영
- CSS스타일
    - 태그에 작성된 스타일 시트 정보 반
<br><br>
